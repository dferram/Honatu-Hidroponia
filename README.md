# Honatu Hidroponia

Tienda online para sistemas de cultivo hidropónico y aeropónico de precisión.

## Descripción

Honatu Hidroponia es una plataforma de e-commerce especializada en sistemas de cultivo vertical, tecnología de iluminación LED, nutrientes especializados y sistemas de monitoreo inteligente para horticultura de precisión.

## Tecnologías

- Next.js 16.2.4
- React 19
- TypeScript 5
- Tailwind CSS 4
- GitHub Pages
